## Сноска

Markdown полезен в разработке[^1].

[^1]: Поддерживается на GitHub, GitLab и в большинстве редакторов кода.

## Alert-блок

> [!TIP]
> Для экранирования специальных символов в Markdown используйте обратный слэш \. Например, чтобы написать *курсив* без форматирования, напишите `\*курсив\*`.

## Escape-символы

- Залоговое без форматирования: \# Это не заголовок
- Звездочка как текст: \*Это просто звёздочки\*
- Нижнее подчёркивание: \_Не курсив\_

# Практическое задание 3

Markdown — это простой язык разметки для быстрого форматирования текста (заголовки, списки, жирный/курсив), который легко преобразуется в HTML и используется для создания документации, постов в блогах и веб-контента, позволяя сосредоточиться на тексте, а не на сложном коде.

---

Основные возможности Markdown — это быстрое и простое форматирование текста (заголовки, списки, ссылки, выделение), сохраняющее читабельность, для последующего преобразования в HTML или другие форматы, идеальное для документации, заметок, README-файлов, блоков и сообщений, с синтаксисом, использующим обычные символы вместо сложного кода, что делает его удобным и интуитивно понятным.

***

**Пример форматирования. Заголовки:**

# Заголовок 1

## Заголовок 2

### Заголовок 3

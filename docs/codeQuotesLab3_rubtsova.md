### 1. Пример команды Git в виде inline-кода
Для просмотра истории коммитов в сокращенном виде используйте команду  
`git log --oneline.`

### 2. Блок кода на языке программирования  
 ```Python
def greet_user(name):
    if name:
        print(f"Привет, {name}!")
    else:
        print("Привет, мир!")

greet_user("Студент")
```
